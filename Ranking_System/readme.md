# Ranking System

**Inputs:**
- Number of player
- ELO rating
- Winner (1/2) for each round or drew(3)

**Output:** <br>
Rank of players with score.

**Rule**
- Swiss Tounament Format is apdoted.
- Drew is 1 point; Win is 2 point; Loss is 0 point.
- Rank according to the Score (and then ELO rating).
